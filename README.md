# 🚀 CryptoChain – Real-Time Crypto Market Website

![CryptoChain Preview](https://github.com/ashishthakur113/CryptoChain1/blob/5c5854bed97de77a4c9d0fe06d18c31914f6e44c/public/cryptoChain%20Website.png)

CryptoChain is a **frontend-focused cryptocurrency market website** built using **React**.  
It fetches **real-time crypto data from a live API**, meaning all prices, market caps, volumes, and charts shown on the website are **100% real and dynamically updated**.

This project is built mainly for **learning, practice, and portfolio purposes**, with a strong focus on **React architecture, API handling, routing, and UI/UX**.

---

## 🚀 Live Demo

🔗 [View the live website](https://crypto-chain1.vercel.app/)

## 📌 Project Overview

CryptoChain provides a clean and modern interface to explore the cryptocurrency market.

### Key Highlights
- Real-time cryptocurrency prices
- Market overview with pagination
- Dedicated coin detail pages
- Interactive price charts
- Demo authentication system (UI only)

---

## ✨ Features

### 🪙 Crypto Market
- Live cryptocurrency prices
- Market cap, volume, 24h change
- Paginated coin listing
- Search coins by name

### 💱 Multi-currency support
- Users can switch between 8 currencies, and all prices update instantly based on the selected currency.


### 📊 Coin Detail Page
- Detailed coin information
- Price, rank, highs & lows
- Interactive price chart (real data)
- External links (Website, Twitter, GitHub, etc.)

### 🔍 Search & Navigation
- Search bar for quick coin lookup
- Smooth client-side routing
- Dedicated routes for each coin

### 🔐 Authentication (Demo)
- Sign Up & Login pages
- Uses **Context API**
- ⚠️ **Note:**  
  This is a **fake / demo authentication system**  
  - No backend
  - No validation
  - Any name/email can log in

---

## 🧭 Pages & Routes

- `/` → Home (Market overview)
- `/coin/:id` → Coin information page
- `/login` → Login (Demo)
- `/signup` → Signup (Demo)
- `/about` → About project
- `/pricing` → Pricing section
